# TP-Link MR6400
## Setup
### Prepare TP-Link MR6400
On the TP-Link
- Insert SIM Card
- Connect to power

On the computer
- Connect to TP-Link via RJ45
- http://192.168.1.1/
- Default user: admin
- Set a password
